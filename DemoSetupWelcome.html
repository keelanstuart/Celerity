This will install a tech demo for Celeritytm, a game / graphics / simulation engine written by Keelan Stuart, Copyright © 2001-2022. All rights reserved.


The controls are simple:

  • A, W, S, D - Moves you around in the world
  • Q, Z - Moves you up and down
  • MOUSE - Looks around
  • MOUSEWHEEL - Zooms in and out, allows orbiting
  • ESC - Toggles your mouse active


Also, now included is C3Edit, the editor application. While not fully functional yet, prototypes are loaded and can be instantiated by double-clicking them. No object selection or manipulation is finished, but the camera can be zoomed and rotated, with the mouse wheel and by holding the shift key while moving the mouse, respectively.
