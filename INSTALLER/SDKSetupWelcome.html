Celerity stands out in the game engine ecosystem by functioning as both a complete engine and a modular framework. Whether you need a full-featured platform or just specific functionality, Celerity adapts to your application's needs. You can use the included C3App or C3Edit applications as starting points for custom software or as the foundation for script- and plug-in-based solutions. Either way, Celerity is designed with intuitive, extensible APIs covering graphics, audio, simulation components, multi-device input mapping, and embedded scripting via a JavaScript-like language.

After the installation is complete:

  • Run SetupCelerityEnv.bat to set the CELERITY_SDK environment variable to the current directory. You may use this to help in setting up projects that include and link Celerity files.
  • Restore the C3Edit default settings by importing C3EditDefaults.reg with regedit

See the Samples directory for, well, samples.

Cheers!
