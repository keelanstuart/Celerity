This will install a tech demo for Celeritytm, a game / graphics / simulation engine written by Keelan Stuart, Copyright © 2001-2024. All rights reserved.


The controls are:

  • A, W, S, D - Moves you around in the world
  • Q, Z - Moves you up and down
  • C - Toggles view mode; track, track position only, free camera
  • MOUSE - Looks around
  • MOUSEWHEEL - Zooms in and out, allows orbiting
  • ESC - Toggles your mouse active


Also included is C3Edit, the editor application. While not fully functional yet, prototypes are loaded and can be instantiated by double-clicking them. Object selection and manipulation is a work in progress, but functional.

The controls are:

  • Camera pan/tilt with the mouse is activated by holding down shift
  • Camera orbit distance is adjusted with the wheel
  • A, W, S, D - Moves you around in the world
  • Q, Z - Moves you up and down
  • C - Centers on selected object(s)
  • ` - Toggles the active axes for the currently selected tool
  • 1, 2, 3, 4, 5 - Activates a tool (Select, Move, Rotate, Uniscale, Scale)
  • Clicking an object will select it. Ctrl-Click will toggle selection (allows group selection)
Controls are cascading... so you can click and drag on an object, moving it around while hold C, so the world appears to move around instead of the object itself... and you can orbit the camera around the object by holding shift. But you never have to let go of the mouse button once you have started an action. Drag to the window edge and the cursor will wrap to the other side, allowing you to continuously move.
I think you'll like it.
